Random Forest Regressor:

The Random Forest Regressor initializes with hyperparameters like the number of trees, maximum depth, and minimum samples to split a node. It employs bootstrapping, creating bootstrap samples for each tree, followed by feature randomization to introduce diversity. Each tree builds upon a bootstrap sample, selecting features for node splits. Predictions from all trees are aggregated, typically through averaging, culminating in the final prediction. This approach yields robust and accurate predictions by leveraging the collective wisdom of multiple decision trees.

XGBoost Regressor:

The XGBoost Regressor begins by initializing hyperparameters like boosting rounds, learning rate, and tree depth. It sequentially builds decision trees to correct errors from previous rounds, optimizing parameters through gradient descent. XGBoost handles imbalanced data by adjusting class imbalance parameters, enhancing performance on minority classes. Once trained, it generates predictions based on learned patterns, offering accurate estimations for test data.

MLP Regressor (Neural Network):

The MLP Regressor is initialized by defining hidden layers, neuron counts, and activation functions. Through forward propagation, inputs traverse the network, with each neuron calculating and applying activations. Errors between predicted and actual outputs are computed and propagated backward, updating neuron weights and biases via gradient descent. Iterative weight and bias updates aim to minimize the loss function, refining predictions. Activation functions like ReLU and sigmoid are specified to introduce non-linearity during initialization.

Model Performance Evaluation:

To effectively assess the models' performance, the historical data underwent partitioning into training and testing sets. The training set was utilized to train the models, while the testing set was employed to validate their accuracy and generalizability. Various metrics, including Mean Squared Error (MSE), Root Mean Squared Error (RMSE), Mean Absolute Percentage Error (MAPE), and R-squared (R2) score, were employed to evaluate the performance of each model. These metrics played a crucial role in determining the models' capability to accurately forecast renewable energy generation.
